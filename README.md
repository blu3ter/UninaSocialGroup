# 🎓 Student Social Network (Simulazione)

Un'applicazione social progettata per facilitare l'interazione tra studenti universitari, permettendo la condivisione di post e lo scambio di opinioni tramite commenti.

---

## 🚀 Funzionalità Principali

* **Gestione Post:** Creazione, visualizzazione e rimozione di contenuti testuali.
* **Interazione:** Sistema di commenti integrato per ogni post.
* **User Experience:** Interfaccia semplice e intuitiva pensata per un contesto accademico.
* **Persistenza:** DB

---

## 🛠️ Stack Tecnologico

Il progetto è stato sviluppato utilizzando le seguenti tecnologie:

| Componente | Tecnologia |
| :--- | :--- |
| **Linguaggio** | Java |
| **Frontend** | Swing |
| **Database** | MySQL |
| **Strumenti** | Maven |
